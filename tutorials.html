From Newbie to Newbro - Tutorials for BRAVE

So you wish to join the Brave Newbies? Well there are a few mandatory steps (like applying to the corp) and handful of recommended steps that you will need to complete in order to join the ranks. We have made a number of Tutorials to help transition from WTF to FAF.

This page is a mirror of the Reddit Post, please check there if a tutorial you want is missing and leave a comment!

In order to get started, you have to Apply to Join the Corp. Once that's completed, it's highly suggested you complete the following:

  • Set Up Jabber (<5 Minutes)
  • Set Up Core (~5-10 Minutes)
  • Set Up Mumble (<5 Minutes, Requires Core)
  • Set Up Push-To-Talk and Whisper for Mumble(<5 Minutes)

Once you've completed the above, you can take a couple of steps to better your experience in EVE. These steps can help you get ships for cheap, escape various camps, and optimize your damage output.

  • How to Optimize your Overview (5 Minutes)

  • How to Find Corporation Ship Fittings (<5 Minutes)
  • How to Buy a Ship via Contracts (<5 Minutes)
  • How to use Tacticals (Better Avoid Gate Camps!) (<5 Minutes)
  • How to Instant-Undock & Dock (Avoid Station Camps!) (<5 Minutes)
  • How to find your Optimal/Falloff (<5 Minutes)
  • How to Use Jump Clones Effectively (<5 Minutes)
